Verilog Basics

This repository contains my Verilog Basics learning notes and practice programs. It covers the fundamental concepts required to start designing digital hardware using Verilog HDL.

Topics Covered

- What is Verilog HDL?
- HDL vs Software Programming
- Module Structure
- Input, Output, and Inout Ports
- Wire and Reg Data Types
- Comments in Verilog
- Numbers and Literals
- Operators
- Continuous Assignment ("assign")
- Basic Coding Examples

Objective

The goal of this repository is to build a strong foundation in Verilog before moving on to combinational logic, sequential logic, RTL design, and advanced VLSI projects.

This repository is part of my VLSI learning journey and will be updated regularly as I progress.
