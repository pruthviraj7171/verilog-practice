# **if else statment**

- In the programing code the computer wil reads the code from top to bottom and in this we dont need every line to get a output.

- so here we use the 'if' , 'else' statment.
  
- In 'if' statment the first we write a programm.(if condition is true then print the program).
  
- If condition false then it directly print the 'elif' statment.
 
- If 'elif' also false it will print else statment.

  ### *FOR EXAMPLE* :- pyton

   age = 20:

       if age < 20
            print("You are above age u are allow for marriage")
       else:
            print("u are under age not eligible for marriage")

 # **while statment**
 
- A while statement is a loop. It tells the computer to repeat a block of code over and over again as long as a specific condition remains True.

- *Think of it like an alarm clock:*

 WHILE the time is not 7:00 AM, keep sleeping. Once it hits 7:00 AM (the condition becomes False), you stop sleeping and wake up.
